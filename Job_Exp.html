In-house DSMC code

This is a part of ongoing work. The code is developed for BCC arrangement of cells for optimal resolution. The code is parallelized using MPI and OpenMP library.

The code is to be validated against following test cases for different Knudsen Number in the range 0.001 to 10.

  • Sod Shock Tube
  • Stationary Shock
  • Couette Flow
  • Poiseuille Flow
  • Thermal Creep
  • Lid Driven Cavity
  • Slider Bearing Problem
  • Flow past micro-geometries

Building in-house particle code for Particulate Flow

This is a part of ongoing work. The particle code is built using C++ to be coupled with in-house Lattice Boltzmann Code for the dilute solution at Stokes Number of 0.5. Two way coupling is considered for the simulation. The codes are parallelized using MPI library.

RANS simulation over Aircraft

RANS (Reynold's Averaged Navier Stokes) has been used in the industry for the prediction of coefficient of Lift and coefficient of Drag and present the L/D characteristic of aeroplane. For this project we used shear stress transport (SST) (k-Ï‰) model.

The client provided the Geometry in .step format which was simplified using FreeCAD to make it suitable for CFD Analysis. Since the Geometry has too many surfaces and Edges, we used the SnappyHexMesh to mesh the Geometry. The simulation of each individual part as well as complete aircraft as a whole was performed using rhoSimpleFoam at Ma 0.3 - 0.7 . For Validation of our setup, we use ONERA M6 and NASA CRM experimental result at Mach No. 0.85. The experimental result and the simulation was in good agreement with each other.

Based on the CFD data, certain design optimization was suggested for better control of aircraft.

Key Notes

  • The coefficient of Drag is very sensitive to the resolution near the boundaries of the aircraft.
  • To match with the experimental coefficient of Drag at high angle of attack, it is quintessential to use the higher order method.
