PDE Based Image Filtering

Course : Mathematical Modeling in Industry
Course Instructor : Dr. S . Sundar
Coding Language : Matlab [ Link ]

As a part of the course project, Perona-Malik, Edge Enhancing Diffusion Model was implemented using Matlab. The contrast parameter lambda plays an important role in the Perona-Malik Model for the preservation of the image boundaries. In order to automatize the contrast parameter lambda, i.e, it can be taken automatically from the image, a new method was conceptualized which sets lambda to be 1.8 times the mean of the gradient after convolving the image. This works well for the test problem. The presentation can be found here

Computational Fluid Dynamics

Course : Advanced Computational Fluid Dynamics
Course Instructor : Dr. Arul K. Prakash
Coding Language: : Matlab [Link ]

In the first phase, the discretization of unsteady state Advection-Diffusion was done using Finite Volume Method. The time marching was done using Crank Nicholson Scheme and space discretization using Upwind and Central Difference Scheme separately. For the convergence speed Gauss Seidel, Conjugate Gradient and Bi-Conjugate Gradient was compared. The detail report can be found here.

In the second phase, the stream-vorticity approach was used to solve the channel flow. The results were compared with the experimental ones.

For the final project, we developed the Volume of Fluid Code (VOF) by Hirt and Nicholas to track the deforming interface. The velocity field was assumed and the deforming interface was tracked accordingly.

Time dependent Heat Conduction Problem on NURBS using Isogeometric Analysis

Course : Isogeometric Analysis
Course Instructor Dr.-Ing. Stefanie Elgeti
Coding Language: : Matlab [ Link ]

During the course project, firstly the Matlab code was developed to construct the B-splines and NURBS along with its derivatives, given the knot vectors. The integration over the NURBS was done using the Gauss Quadrature rule. The NURB refinement code was provided during the course. Using the concept of IsoGeometric Analysis the unsteady heat conduction problem was solved on the NURBS.
