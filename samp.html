  • Building inhouse CAD tool

    For Latticle Bolzman method, the BCC grid is found to be the optimal. The current CAD tool designs NURBS on the Cartesian grid. The current effort is geared towards effective construction of NURBS on BCC grid.

    Building inhouse particle code for Granular Flow and DSMC

    This is a part of ongoing work. The particle code is built using C++ to be coupled with inhouse Lattice Bolzman Code for the dilute solution at Stokes Number of 0.5. Two way coupling is considered for the simulation. The code is to be further exended to perform effecient DSMC simulation. All the codes are parallelized using MPI library.
